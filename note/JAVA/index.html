JAVA从入门到精通

约 792 字大约 3 分钟

2026-03-02

简介

Java优点：

  • Java是基于JVM虚拟机的跨平台语言，一次编写，到处运行；
  • Java程序易于编写，而且有内置垃圾收集，不必考虑内存管理；
  • Java虚拟机拥有工业级的稳定性和高度优化的性能，且经过了长时期的考验；
  • Java拥有最广泛的开源社区支持，各种高质量组件随时可用。

Java语言常年霸占着三大市场：

  • 互联网和企业应用，这是Java EE的长期优势和市场地位；
  • 大数据平台，主要有Hadoop、Spark、Flink等，他们都是Java或Scala（种运行于JVM的编程语言）开发的；
  • Android移动平台。

Java分为三个不同版本：

  • Java SE：Standard Edition 标准版，包含标准的JVM和标准库
  • Java EE：Enterprise Edition 企业版，在Java SE的基础上加上了大量的API和库，以便方便开发Web应用、数据库、消息服务等
  • Java ME：Micro Edition 它是一个针对嵌入式设备的“瘦身版”，Java SE的标准库无法在Java ME上使用，Java ME的虚拟机也是“瘦身版”

推荐的Java学习路线图如下：

  1. 首先要学习Java SE，掌握Java语言本身、Java核心开发技术以及Java标准库的使用；
  2. 如果继续学习Java EE，那么Spring框架、数据库开发、分布式架构就是需要学习的；
  3. 如果要学习大数据开发，那么Hadoop、Spark、Flink这些大数据平台就是需要学习的，他们都基于Java或Scala开发；
  4. 如果想要学习移动开发，那么就深入Android平台，掌握Android App开发。

搭建开发环境

当你用记事本编写的时候，如果没有保存内容它就会消失掉，也就是说它是暂时存储的，速度很快，当点击保存以后，我们编写的内容就会被写入到硬盘当中。

什么是内存地址

比如在运行程序时，需要临时储存一些数据，为了能够快速找到该数据，我们就会给每个内存空间进行编号，这就是内存地址。

JAVA中的内存分配

java在运行的时候会占用一部分内存，它将这些内存分为这些区域:栈内存、堆内存、方法区、本地方法区、程序计数器

栈内存：每个线程都自己独立的栈，目前只要考虑一个即可，方法被调用进栈执行，执行完毕出栈。

堆内存：所有线程共享，存储对象、数组、字符串常量池。

方法区：JDK永久代实现，JDK8+元空间实现，从虚拟机内部移到本地内存。存储字节码信息、常量、静态变量。

本地方法区：调用本地Native方法

程序计数器：每个线程独立、记录当前线程执行的字节码指令地址(行号)

基本数据类型：变量里面记录的是真实的数据据，传递的也是真实的数据
