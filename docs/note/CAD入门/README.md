2.滚动鼠标滚轮可以放大缩小

3.范围缩放，键盘输入z+空格或者点击右边的工具栏范围缩放,回到图像的正中央。

4.小抓手  键盘输入p+空格。或者右边工具栏点击小抓手。

5.画直线

点击直线 或者按下L+ 空格键，鼠标左击一下，为一个断点。按空格完成本次绘画。如果希望直线水平竖直，则可以按下F8打开正交。输入C 能让线首尾相连

6.画圆

点击园 或者 按下C + 空格键 ，如果要切换到直径 按下D 输入值.

三点相切 首先输入c + 空格 输入3p 输入 tan 点击一条边 循环三次

7.圆弧

满足三条点 输入tan+空格

8.多段线

有多条线段组成，可以绘制圆弧。键盘输入pl + 空格 进入多段线模式，一开始默认画的是直线，输入a +空格 可以绘制圆弧

将分开的线连在一起 输入PE + 空格 然后 输入  M 然后选择 Y 然后合并
